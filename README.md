# tvencoder

How to install
--------------------

Make a virtualenv

```
virtualenv ~/env/encode
cd ~/env/encode
source bin/activate
```

Install packages

```
pip install -r requiments.txt
```

Add enqueue.sh to your Chinachu

Example:

```
  "recordedCommand": "/home/chinachu/src/tvencoder/enqueue.sh",
```
